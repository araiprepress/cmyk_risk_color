# 画像のリスクカラー警告ツール

画像をアップロードすると、CMYK変換時に色が大きく変化する可能性があるリスクカラーを検出し、警告表示するWebアプリケーションです。

## 機能

- ドラッグ&ドロップで画像をアップロード
- リスクカラーの自動検出
- リスクカラーを赤で表示した警告画像の生成
- リスクカラーを安全な色に変換した画像の生成
- 画像のダウンロード機能

## 使い方

1. `index.html`をブラウザで開く
2. 画像をドラッグ&ドロップエリアにドロップ
3. リスクカラーの検出結果が表示されます
4. 必要に応じて警告画像や変換後の画像をダウンロード

## ファイル構成

- `index.html` - メインのHTMLファイル
- `style.css` - スタイルシート
- `dd_img.js` - ドラッグ&ドロップ機能
- `risk_color_detector.js` - リスクカラー検出ロジック
- `risky_colors.csv` - リスクカラーのデータベース

## 技術スタック

- HTML5
- CSS3
- JavaScript (Vanilla)
- Canvas API
